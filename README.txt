Hello, this is the assignment 1(2) of SDF course. 
This terminal is made using C language without using system calls, etc.


TO START THE TERMINAL:
1) Open the folder in terminal.
2) Type "make" in the terminal.
3) Type "./myShell" in the terminal.



SALIENT FEATURES OF THE TERMINAL:
1) Has similar feel as the real terminal.
2) The up and down arrow keys work, for see the recent commands.
3) There is tab completion in the commands wherever necessary.
4) exit command to exit the terminal anytime.
5) myman command for giving some information about the commands.
6) Colours in the dirname, hostname and currdir.



THIS TERMINAL SUPPORTS THESE COMMANDS:
1) myls - similar to ls in original terminal.
    Flags supported - l,S,r,a,t and all logical combinations(all doing similar functionality to the original terminal).
2) myps - similar to ps in original terminal.
    Flags supported - e,a,F,f and all logical combinations(all doing similar functionality to the original terminal).
3) mycp - similar to cp in original terminal.
    Flags supported - I,n,u,v and all logical combinations(all doing similar functionality to the original terminal).
4) mymv - similar to mv in original terminal.
    Flags supported - I,n,u,v and all logical combinations(all doing similar functionality to the original terminal).
5) mygrep - similar to grep in original terminal.
    Flags supported - I,c,n,w and all logical combinations(all doing similar functionality to the original terminal).
6) cd - similar to cd in original terminal.
7) exit - to exit the terminal
8) myman - similar to man in original terminal. It gives some basic information about the above 7 commands.
